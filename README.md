# Procedimiento
## Drawer ( con 4 opciones)
Encabezado: imagen (yo pongo el draw) desde loa red, nombre empresa (STEAM), dirección, teléfono, correo, procura que la información no se sobreponga al logo o el resto del darwer, utiliza colores llamativos como los de STEAM, azul marino, etc...

Listile: icono (etiqueta de compras, llave inglesa, usuario, </>), texto y acción para abrir  pagina según opción de listile, (productos, técnicos, usuarios, desarrolladores...)

En cada pagina una imagen centrada de 200x200 desde la red  (desde github, yo mismo coloco los raw)

Utilizar rutas nombradas para navegar entre paginas desde el drawer (rutas nombradas. trabajar con firebase studio)
##
Probar en dartpad antes de trabajar en firbase studio.
en firebase studio el main.dart  rutas nombradas + carpeta  "LasPaginas" (Procura NO EQUIVOCARTE en este paso, es "LasPaginas", nada mas ni nada menos) y dentro de ellas los archivos correspondientes de dart.
Debes darme el codigo de cada pagina no un ejemplo
